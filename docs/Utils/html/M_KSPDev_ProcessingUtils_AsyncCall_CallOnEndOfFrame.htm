AsyncCallCallOnEndOfFrame Method

Delays execution of the delegate till the end of the current frame.

Namespace:  KSPDev.ProcessingUtils
Assembly:  KSPDev_Utils.0.24 (in KSPDev_Utils.0.24.dll) Version: 0.24 for KSP v1.3

	MonoBehaviour mono,
	Action action
)
Request Example View Source

Parameters

mono
Type: UnityEngineMonoBehaviour
Unity object to run coroutine on. If this object dies then the async call will not be invoked.
action
Type: SystemAction
Delegate to execute.

Return Value

Type: Coroutine
Coroutine instance.
Remarks
Caller can continue executing its logic. The delegate will be called at the end of the frame via Unity StartCoroutine mechanism. The delegate will be called only once.
Examples
Copy
class MyComponent : MonoBehaviour {
  void Update() {
    // Call the method at the end of the current frame.
    AsyncCall.CallOnEndOfFrame(this, () => Debug.LogFormat("Async call!"));
  }
}
