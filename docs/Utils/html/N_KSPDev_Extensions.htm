KSPDev.Extensions Namespace

A set of handy extensions to the existing C# and KSP classes. Normally, you never call them from this assembly. Instead, you get them at the instance of the affected class.

Examples
For instance, here is how extension to standard C# Dictionary can be used:
Copy
var a = new Dictionary<int, List<int>>;
a.SetDefault(0).Add(1);  // Creates empty list and adds it as entry 0.
a.SetDefault(0).Add(2);  // Re-uses existing entry 0.
